# aml-vscode README

Internal mulesoft extension to play arround with ALS server. As ALS uses AMF framework, you also will be able to test AMF validations.

## Supported Web Apis:
* Raml 1.0
* Raml 0.8
* Oas 2.0
* OpenApi 3.0.0
* AML instance documents: beta

## Features

* Completion
* Diagnostics
* Structure
* Document Links
* Goto Definition
* Find Uses (beta)

## Requirements

Java 1.8 or later
